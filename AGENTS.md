# Project rules

- Work only in the `main` branch. Do not create new branches.
- Before making the first edit in a session, synchronize local files with the remote `main` exactly once using `git pull --ff-only origin main`, then work from the latest commit.
- Because work alternates with Claude Code, treat the remote `main` branch as the source of truth before edits.
- After every change, immediately run `git add`, create a clear commit, and push to `origin main`.
