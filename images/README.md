# Фото Дианы для сайта

Сейчас вместо реальных фото показываются стилизованные SVG-портреты
(они вшиты прямо в HTML как фолбэк). Чтобы заменить на настоящие фото,
загрузите два файла с этими именами:

| Имя файла          | Откуда брать                                  | Где появится                            |
|--------------------|-----------------------------------------------|-----------------------------------------|
| `diana-hero.jpg`   | На фоне синей стены, в джинсовке              | Hero + первая карточка в «Поближе»      |
| `diana-about.jpg`  | В кардигане + полосатой майке, светлый фон    | «Обо мне» + третья карточка в «Поближе» |

Форматы: `.jpg` или `.png`. Рекомендуемый размер — 1200–1600 px по длинной стороне.

Как только файлы окажутся в этой папке, реальные фото автоматически
перекроют SVG-портреты — без правок в HTML или CSS.
