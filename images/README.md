# Фото Дианы для сайта

Положите 4 фото с такими именами:

| Имя файла          | Какое фото                                | Где появится                     |
|--------------------|-------------------------------------------|----------------------------------|
| `diana-hero.jpg`   | На фоне синей стены, в джинсовке          | Hero (главный экран)             |
| `diana-about.jpg`  | В кардигане, тёплый светлый фон, улыбка   | Секция «Обо мне»                 |
| `diana-walk.jpg`   | На пешеходном переходе, пальмы            | Коллаж в секции «Поближе»        |
| `diana-red.jpg`    | В красном платье, тёмный фон              | Коллаж в секции «Поближе»        |

Форматы: `.jpg` или `.png` подойдут. Рекомендованный размер по длинной стороне — 1200–1600 px.
